# Codewars Solutions (Go)

This repository contains my solutions to various Katas on [Codewars](https://www.codewars.com), implemented in Go.

## Project Structure

- **solutions/**: Contains the source code and test files for each solved Kata.
- **go.mod**: Go module definition.

## Running Tests

To run all tests in the project:

```bash
go test -v ./...
```

To run tests for a specific file/package, navigate to `solutions/` or target the specific package.
